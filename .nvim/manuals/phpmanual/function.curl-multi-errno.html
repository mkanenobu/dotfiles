curl_multi_errno

(PHP 7 >= 7.1.0)

curl_multi_errno — Return the last multi curl error number

説明

int curl_multi_errno ( resource $mh )

Return an integer containing the last multi curl error number.

パラメータ

mh

curl_multi_init() が返す cURL マルチハンドル。

返り値

Return an integer containing the last multi curl error number, 失敗した場合に FALSE を返します.
