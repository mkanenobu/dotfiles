stats_stat_powersum

(PECL stats >= 1.0.0)

stats_stat_powersum — ベクトルのべき乗和を返す

説明

float stats_stat_powersum ( array $arr , float $power )

配列 arr として表されたベクトルの power 乗和を返します。

パラメータ

arr

入力配列

power

べき数

返り値

入力配列のべき乗和を返します。
