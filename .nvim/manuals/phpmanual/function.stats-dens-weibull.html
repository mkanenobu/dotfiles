stats_dens_weibull

(PECL stats >= 1.0.0)

stats_dens_weibull — ワイブル分布の確率密度関数

説明

float stats_dens_weibull ( float $x , float $a , float $b )

形状母数 a, 尺度母数 b のワイブル分布にしたがう確率変数の x における確率密度を返します。

パラメータ

x

確率変数の値

a

形状母数

b

尺度母数

返り値

x における確率密度。失敗時は FALSE を返します。
