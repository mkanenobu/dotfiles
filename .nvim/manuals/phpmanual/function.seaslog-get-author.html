seaslog_get_author

(PECL seaslog >=1.0.0)

seaslog_get_author — Get SeasLog author.

説明

string seaslog_get_author ( void )

パラメータ

この関数にはパラメータはありません。

返り値

Return SeasLog author as string.

例

例1 seaslog_get_author() example

<?php

var_dump(seaslog_get_author());

?>

上の例の出力は、 たとえば以下のようになります。

string(29) "Chitao.Gao  [ neeke@php.net ]"
