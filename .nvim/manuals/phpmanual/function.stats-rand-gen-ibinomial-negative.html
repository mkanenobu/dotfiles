stats_rand_gen_ibinomial_negative

(PECL stats >= 1.0.0)

stats_rand_gen_ibinomial_negative — 負の二項分布から無作為な値を生成する

説明

int stats_rand_gen_ibinomial_negative ( int $n , float $p )

成功回数 n, 成功確率 p の負の二項分布からの無作為な値を返します。

パラメータ

n

成功回数

p

成功確率

返り値

無作為な値。失敗回数を表します。
