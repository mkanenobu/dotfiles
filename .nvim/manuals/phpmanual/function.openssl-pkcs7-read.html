(PHP 7 >= 7.2.0)

openssl_pkcs7_read — Export the PKCS7 file to an array of PEM certificates

説明

bool openssl_pkcs7_read ( string $infilename , array &$certs )

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。
