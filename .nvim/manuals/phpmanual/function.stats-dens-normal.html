stats_dens_normal

(PECL stats >= 1.0.0)

stats_dens_normal — 正規分布の確率密度関数

説明

float stats_dens_normal ( float $x , float $ave , float $stdev )

平均 ave, 標準偏差 stdev の正規分布にしたがう確率変数の x における確率密度を返します。

パラメータ

x

確率変数の値

ave

平均

stdev

標準偏差

返り値

x における確率密度。失敗時は FALSE を返します。
