pg_consume_input

(PHP 5 >= 5.6.0, PHP 7)

pg_consume_input — 接続の入力を読み取る

説明

bool pg_consume_input ( resource $connection )

pg_consume_input() は、データベースサーバーからの 読み取りを待機中のすべての入力を消費します。

パラメータ

connection

PostgreSQL の接続リソース。

返り値

エラーが発生しなかった場合 TRUE、 またはエラーがあった場合 FALSE。 TRUE は、入力が読み取り待機中ということを示すとは限らないことに注意してください。
