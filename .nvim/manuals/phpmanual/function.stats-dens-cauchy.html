stats_dens_cauchy

(PECL stats >= 1.0.0)

stats_dens_cauchy — コーシー分布の確率密度関数

説明

float stats_dens_cauchy ( float $x , float $ave , float $stdev )

位置母数 ave, 尺度母数 stdev のコーシー分布にしたがう確率変数の x における確率密度を返します。

パラメータ

x

確率変数の値

ave

位置母数

stdev

尺度母数

返り値

x における確率密度。失敗時は FALSE を返します。
