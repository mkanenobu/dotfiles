(PHP 7 >= 7.2.0)

mb_ord — Get code point of character

説明

int mb_ord ( string $str [, string $encoding ] )

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

str

encoding

返り値

Returns a code point of character失敗した場合に FALSE を返します.
