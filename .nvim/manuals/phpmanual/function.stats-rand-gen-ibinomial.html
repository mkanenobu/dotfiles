stats_rand_gen_ibinomial

(PECL stats >= 1.0.0)

stats_rand_gen_ibinomial — 二項分布から無作為な値を生成する

説明

int stats_rand_gen_ibinomial ( int $n , float $pp )

試行回数を n, 各試行で事象の発生する確率を pp とする二項分布からの無作為な値を返します。

パラメータ

n

試行回数

pp

各試行で事象の発生する確率

返り値

無作為な値
