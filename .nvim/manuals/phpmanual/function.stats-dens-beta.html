stats_dens_beta

(PECL stats >= 1.0.0)

stats_dens_beta — ベータ分布の確率密度関数

説明

float stats_dens_beta ( float $x , float $a , float $b )

形状母数 a, b のベータ分布にしたがう確率変数の x における確率密度を返します。

パラメータ

x

確率変数の値

a

形状母数

b

形状母数

返り値

x における確率密度。失敗時は FALSE を返します。
